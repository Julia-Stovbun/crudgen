# Contributing

Thanks for helping improve crudgen! This guide keeps contributions simple and consistent.

## Setup
- Use Python 3.11+ (3.12 is also supported in CI).
- Install dev dependencies: `pip install -e ".[dev]"`.

## Testing
- Run the test suite: `pytest`.

## Formatting & linting
- Check style: `ruff check .`.
- Keep changes minimal to preserve the lite scope.
